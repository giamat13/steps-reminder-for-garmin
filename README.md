# Steps Reminder Widget - תזכורת צעדים

Widget לשעוני Garmin (Fenix 7 Pro) שמתריע כאשר המשתמש מפגר מיעד הצעדים היומי.

## תכונות

### לוגיקה בסיסית
- בדיקה אוטומטית כל 5 דקות (ברקע)
- השוואה בין התקדמות היום להתקדמות הצעדים
- התראה באמצעות רטט וצליל

### שני מצבי עבודה

#### 1. **מצב אחוזים** (Percentage Mode)
- **סף זמן (Time Threshold)**: אחוז מהיום (0-100)
- **סף צעדים (Steps Threshold)**: אחוז מיעד הצעדים (0-100)
- **דוגמה**: אם עברו 50% מהיום אך השלמת רק 30% מהצעדים → התראה

#### 2. **מצב מוחלט** (Absolute Mode)
- **סף זמן**: דקות מחצות (0-1440)
  - דוגמה: 720 = 12:00 בצהריים
- **סף צעדים**: מספר מוחלט של צעדים
- **דוגמה**: אם השעה 14:00 (840 דקות) ועשית רק 3000 צעדים → התראה

## הגדרות

גש ל-Connect IQ Settings בשעון כדי להגדיר:

1. **Use Percentage Mode** - בחר בין מצב אחוזים (true) למצב מוחלט (false)
2. **Time Threshold** - סף הזמן (אחוזים או דקות, תלוי במצב)
3. **Steps Threshold** - סף הצעדים (אחוזים או מספר מוחלט, תלוי במצב)

### ערכי ברירת מחדל
- מצב: אחוזים
- סף זמן: 50%
- סף צעדים: 50%

## התקנה ובנייה

### דרישות מקדימות
- Garmin Connect IQ SDK 7.4.2+
- VSCode עם Monkey C Extension
- Fenix 7 Pro (או סימולטור)

### שלבים
1. פתח את התיקייה ב-VSCode
2. התחבר לשעון או לסימולטור
3. לחץ Ctrl+Shift+P → "Monkey C: Build for Device"
4. לחץ Ctrl+Shift+P → "Monkey C: Run"

## API ותאימות

- **API Level**: 5.2.0+
- **Target Device**: Fenix 7 Pro
- **הרשאות נדרשות**:
  - `Background` - רקע לבדיקות תקופתיות
  - `ActivityInfo` - גישה לנתוני צעדים
  - `Sensor` - גישה לחיישנים
  - `Attention` - רטט והתראות

## טיפים לשימוש

### דוגמאות הגדרה

**למי שרוצה להישאר במסלול כל היום:**
- מצב: אחוזים
- סף זמן: 50%
- סף צעדים: 50%

**למי שרוצה להתריע אם לא עשה 5000 צעדים עד הצהריים:**
- מצב: מוחלט
- סף זמן: 720 (12:00)
- סף צעדים: 5000

**למי שעובד משרדי (רוצה התראה בשעה 16:00 אם לא עשה 40% מהיעד):**
- מצב: מוחלט
- סף זמן: 960 (16:00)
- סף צעדים: 40% מיעד יומי (צריך לחשב ידנית)

## פתרון בעיות

### אין התראות
1. בדוק שההרשאות אושרו בהגדרות השעון
2. ודא ש-Background מופעל
3. בדוק שיעד הצעדים היומי מוגדר בפרופיל המשתמש

### נתונים לא מוצגים
- ודא שהשעון מקבל נתוני צעדים (בדוק ב-Activity Monitoring)
- הפעל מחדש את ה-Widget
